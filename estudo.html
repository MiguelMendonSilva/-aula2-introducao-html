Filmes

html
Onde ficam todos os elementos da pagina
head
Onde ficam tudo o que não é conteúdo
body
Onde fica o que aparece na pagina
style
Onde ficam todo o estilo de layout da pagina
script
Onde fica todo o js da pagina
h1
Forma o texto para um formato de título
p
Define uma parte do texto que será um parágrafo
br
Uma quebra de linha
img
Representa a inserção de uma imagem
ul
Uma lista não ordenada
ol
Uma lista ordenada
li
Representa os itens que fazem parte de uma lista
dt
Identifica um termo na lista de definição
dl
Uma lista de termos e descrições
dd
Detalhes ou definição do termo
DOCTYPE
Indica o tipo do documento para o navegador
